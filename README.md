# 🧠 Cyber Lab Portfolio

A futuristic cybersecurity portfolio designed as an interactive cyber investigation lab.


## 🚀 Features

- 🔍 OSINT Investigation Module  
- 🧪 Digital Forensics Lab  
- 🌐 Network Traffic Analysis Simulation  
- 💀 Exploit Demonstrations (SQL Injection, XSS - safe environment)  
- ⚡ Interactive Cyber UI (Neon + Terminal Theme)  
- 🎬 Smooth animations and transitions  


## 🛠️ Tech Stack

- React (Vite)  
- Tailwind CSS  
- Framer Motion  
- Three.js  


## 📂 Project Structure

- Modular component-based architecture  
- Reusable UI components  
- Scalable and maintainable codebase  

## 📌 About

This project showcases my skills in cybersecurity concepts, frontend development, and interactive UI design through a lab-style interface.

## 🔐 Disclaimer

All demonstrations and simulations in this project are created for educational purposes only.
